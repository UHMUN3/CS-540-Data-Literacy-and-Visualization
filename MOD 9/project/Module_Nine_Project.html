Data Quality Observations¶
Based on the exploratory analysis above, several important characteristics emerge:


Structural observations:


    

  • All three data sets are in long format — each row represents a single attribute/value pair for one geographic area. We will need to pivot them to wide format before merging.
    • 

  • The three files use inconsistent column names for the same conceptual fields:
      

    • FIPS identifier: FIPS Code (Education) vs FIPS_Code (Unemployment) vs FIPStxt (Population)
      • 

    • Area label: Area name (Education) vs Area_Name (Unemployment and Population)
      • 

    

    • 

  • We will need to standardize these column names before blending.
    • 



Coverage observations:


    

  • The Education file contains educational attainment estimates pooled over the 2019–23 period (8 attributes: 4 raw counts + 4 percentages).
    • 

  • The Unemployment file covers a 5-year span (2019–2023) with annual labor-force, employment, unemployment, and rate values, plus 2022 median household income.
    • 

  • The Population file covers a 4-year span (2020–2023) with births, deaths, migration, and population estimates.
    • 

  • All three files operate at the state level (approximately 54 areas: U.S. total + 50 states + DC + territories).
    • 



Quality observations:


    

  • Missing values are modest in scale and concentrated in specific attributes; we will handle them during preprocessing.
    • 

  • No duplicated rows were detected.
    • 

  • The Value column mixes numeric counts (e.g., 79,954,302 adults with a bachelor's degree) and percentages (e.g., 35.0% bachelor's-or-higher rate). After we pivot to wide format, each attribute becomes its own column and these units no longer collide.
    • 



These observations directly shape the preprocessing strategy in the next section.

# ---------------------------------------------------------------------------
# Step 1: Standardize column names so the three sources share a common key
# ---------------------------------------------------------------------------
education_df = education_df.rename(columns={
    "FIPS Code": "FIPS",
    "Area name": "Area_Name"
})

unemployment_df = unemployment_df.rename(columns={
    "FIPS_Code": "FIPS"
})

population_df = population_df.rename(columns={
    "FIPStxt": "FIPS"
})

# ---------------------------------------------------------------------------
# Step 2: Pivot each long-format data set into wide format
# ---------------------------------------------------------------------------
# After pivoting, each row will represent one geographic area and each
# Attribute becomes its own column.

def pivot_to_wide(df):
    wide = df.pivot_table(
        index=["FIPS", "State", "Area_Name"],
        columns="Attribute",
        values="Value",
        aggfunc="first"
    ).reset_index()
    wide.columns.name = None   # drop the leftover "Attribute" axis label
    return wide

education_wide    = pivot_to_wide(education_df)
unemployment_wide = pivot_to_wide(unemployment_df)
population_wide   = pivot_to_wide(population_df)

print(f"Education (wide):    {education_wide.shape}")
print(f"Unemployment (wide): {unemployment_wide.shape}")
print(f"Population (wide):   {population_wide.shape}")

# Step 4: Handle missing values and engineer features for the analysis
# ---------------------------------------------------------------------------

# Check missing values in the merged data
missing_summary = merged_df.isnull().sum().sort_values(ascending=False)
print("Columns with the most missing values (top 10):")
print(missing_summary[missing_summary > 0].head(10).to_string())

# Create a state-level frame (drop the U.S. national row, FIPS == 0)
states_df = merged_df[merged_df["FIPS"] != 0].copy()

# --- Engineered features ---
# Population growth from 2020 census to 2023 estimate (%)
states_df["Pop_Change_Pct_2020_2023"] = (
    (states_df["POP_ESTIMATE_2023"] - states_df["CENSUS_2020_POP"])
    / states_df["CENSUS_2020_POP"] * 100
)

# Change in unemployment rate from 2019 (pre-pandemic) to 2023
states_df["Unemployment_Change_2019_2023"] = (
    states_df["Unemployment_rate_2023"] - states_df["Unemployment_rate_2019"]
)

# Composite "Expansion Attractiveness" score for the executive audience.
# Higher bachelor's-degree %, lower unemployment, and positive population
# growth all push the score upward. Each input is min-max normalized.
def normalize(series):
    return (series - series.min()) / (series.max() - series.min())

states_df["Expansion_Score"] = (
    normalize(states_df["Percent of adults with a bachelor's degree or higher, 2019-23"])
    + normalize(-states_df["Unemployment_rate_2023"])
    + normalize(states_df["Pop_Change_Pct_2020_2023"])
) / 3

print(f"\nState-level frame shape: {states_df.shape}")

# AUDIENCE 1: Corporate Executives looking to expand
# ----------------------------------------------------------------------------
# High-level overview — a ranked horizontal bar chart of the top 15 states 
# by composite Expansion_Score. Bar color encodes 2022 median household 
# income, adding a market-purchasing-power dimension on top of the ranking.
# Two attributes are integrated in a single, scannable visualization.
# ============================================================================

# Build the top-15 frame
top_states = (
    states_df
    .dropna(subset=["Expansion_Score", "Median_Household_Income_2022"])
    .nlargest(15, "Expansion_Score")
    .sort_values("Expansion_Score")    # ascending so the leader appears at the top
)

fig, ax = plt.subplots(figsize=(12, 8))

# Color each bar by 2022 median household income (a separate attribute from the score)
norm = plt.Normalize(
    vmin=top_states["Median_Household_Income_2022"].min(),
    vmax=top_states["Median_Household_Income_2022"].max()
)
colors = plt.cm.YlGnBu(norm(top_states["Median_Household_Income_2022"]))

bars = ax.barh(
    top_states["Area_Name"],
    top_states["Expansion_Score"],
    color=colors, edgecolor="white", linewidth=0.6
)

# Place the numeric score at the end of each bar
for bar, score in zip(bars, top_states["Expansion_Score"]):
    ax.text(
        bar.get_width() + 0.01,
        bar.get_y() + bar.get_height() / 2,
        f"{score:.2f}",
        va="center", fontsize=10, fontweight="bold"
    )

# Colorbar to explain the color dimension
sm = plt.cm.ScalarMappable(cmap="YlGnBu", norm=norm)
sm.set_array([])
cbar = plt.colorbar(sm, ax=ax, shrink=0.65, pad=0.02)
cbar.set_label("Median Household Income, 2022 ($)", fontsize=10)

ax.set_xlabel("Expansion Attractiveness Score (0 = lowest, 1 = highest)", fontsize=11)
ax.set_title(
    "Top 15 States for Business Expansion (2023)\n"
    "Composite of education, low unemployment, and population growth — colored by household income",
    fontsize=14, fontweight="bold", pad=15
)
ax.set_xlim(0, top_states["Expansion_Score"].max() * 1.15)
ax.spines["top"].set_visible(False)
ax.spines["right"].set_visible(False)

plt.tight_layout()
plt.show()

Key Insights from the Visualizations¶
High-level overview (corporate-executive audience). The composite Expansion Attractiveness Score ranks states by combining three labor-market factors: educated workforce, low unemployment, and population growth. The top of the ranking is dominated by states with strong knowledge-economy footprints and net inbound migration. Adding median household income as the color dimension lets executives instantly see not only which states rank highly but also how strong the local consumer market is — a critical second-order consideration when choosing an expansion site. The simple bar layout and clear numeric labels keep the visualization scannable in seconds, which fits the audience's need for a top-line summary.


Detailed insight (county-commissioner audience). The four-dimensional bubble scatter reveals a clear diagonal pattern: states with higher educational attainment tend to have lower unemployment rates. The bottom-right "target zone" contains the strongest combinations. Bubble color adds a forward-looking dimension — many of the bottom-right states are also growing in population (green), suggesting these advantages may compound over time. The quadrant labels and median reference lines give commissioners an immediate sense of where their state sits relative to peers across multiple dimensions, and where the workforce-development gaps are most concentrated.


Cross-cutting takeaway. Education attainment emerges as the single strongest correlate with both low unemployment and population growth. For corporate executives, this signals where the talent pool will be. For commissioners, it signals where the most leveraged local-policy investments lie — typically in post-secondary education and workforce-training pipelines.
